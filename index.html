PROJECTS

PENCIL PUSHERS

A project we built during Hack @ Brown 2017. It's a local co-op fast-paced competitive brawler where you have a javelin like pencil you must throw to kill the opponent. You each have 5 lives. Go ahead, play it for yourself by clicking the link above.

OBAMA | MARKOV

A web interface for a Markov chain that utilizes Obama's past speeches. We built the Obama chain as a side project during Bronco's tackleSTEM.

CUBE RUNNER

A project I built during Local Hack Day. It's a side-scrolling endless platformer that gradually gets faster the longer the game goes on. You gain points by maintaining contact with platforms. At the end, the top 5 leaderboards are displayed. It was chosen as a Devpost staff pick, and featured on their newsletter.

HER
A project we built at Bronco's tackleSTEM. We built an Alexa-skill which integrates with the CleverBot API, so that we could maintain a coherent conversation whenever we prompted "Her".
